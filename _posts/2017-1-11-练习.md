###### 从键盘接收一个文件夹路径,统计该文件夹大小
```java
public static void main(String[] args) {
    Scanner sc=new Scanner(System.in);
    System.out.println("请输入文件夹路径");
    File dir;
    while(true){
        String line = sc.nextLine();
        dir=new File(line);
        if(!dir.exists())
            System.out.println("您输入的文件路径不存在");
        else if(dir.isFile())
            System.out.println("您输入的是文件路径，请输入文件夹路径");
        else
            break;
    }
    System.out.println(getDirSize(dir));
}
public static long getDirSize(File dir){
    long sum=0;
    File[] files = dir.listFiles();
    for(File subFile:files){
        if(subFile.isFile())
            sum+=subFile.length();
        else
            sum+=getDirSize(subFile);
    }
    return sum;
}
```

###### 递归删除文件夹
```java
public static void deleteDir(File dir){
    File[] files = dir.listFiles();
    for(File subFile:files){
        if(subFile.isFile())
            subFile.delete();
        else
            deleteDir(subFile);
    }
    dir.delete();
}
```

###### 从键盘接收两个文件夹路径,把其中一个文件夹中(包含内容)拷贝到另一个文件夹中
```java
public static void main(String[] args) throws Exception {
   File src=Test1.getDir();
   File dest=Test1.getDir();
   if(src.equals(dest)){
       System.out.println("目标文件是源文件子类");
   }else {
       copy(src,dest);
   }
}
public static void copy(File src,File dest)throws  IOException{
    File dir = new File(dest, src.getName());
    dir.mkdirs();
    File[] files = src.listFiles();
    for(File subFile:files){
        if(subFile.isFile()){
            BufferedInputStream bis=new BufferedInputStream(new FileInputStream(subFile));
            BufferedOutputStream bos=new BufferedOutputStream(new FileOutputStream(new File(dir,subFile.getName())));
            int b;
            while((b=bis.read())!=-1){
                bos.write(b);
            }
            bis.close();
            bos.close();
        }else {
            copy(subFile,dir);
        }
    }
}
```

###### 从键盘接收一个文件夹路径按层级打印
```java
public static void printLevel(File dir,int level){
    File[] files = dir.listFiles();
    for(File subFile:files){
        for(int i=0;i<level;i++){
            System.out.print("\t");
        }
        System.out.println(subFile.getName());
        if(subFile.isDirectory())
            printLevel(subFile,level+1);
    }
}
```

###### 斐波那契数列不死神兔,从第三个开始等于前面2个之和
```java
public static int getNum(int num){
    if(num==1||num==2){
        return 1;
    }
    return getNum(num-1)+getNum(num-2);
}
public static int getNumbyArray(int num){
    int[]arr=new int[num];
    arr[0]=1;
    arr[1]=1;
    for(int i=2;i<num;i++){
        arr[i]=arr[i-1]+arr[i-2];
    }
    return arr[num-1];
}
```

###### 约瑟夫环
```java
public static int getLacklyNum(int num){
    ArrayList<Integer> list = new ArrayList<>();
    for(int i=1;i<=num;i++){
        list.add(i);
    }
    int count=1;
    for(int i=0;list.size()!=1;i++){
        if(i==list.size()){
            i=0;
        }
        if(count++%3==0){
            list.remove(i--);
```
